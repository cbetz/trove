# web

Static visualization site built with [Observable Framework](https://observablehq.com/framework), querying the published Parquet bundles via DuckDB-WASM in the browser.

**Status:** not yet scaffolded. See milestone M6.

Planned pages:

- Hospital profile (financials, trends, peer compare)
- State choropleth (operating margin, bed capacity)
- Hospital reporting lookup
- Field glossary (searchable)
