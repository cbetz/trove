# crosswalk

CCN ↔ EIN crosswalk linking CMS Medicare Cost Reports (keyed by CCN) to IRS Form 990 filings (keyed by EIN).

Seeded from Sacarny's `hospital-cost-reports` repo and Community Benefit Insight, with an agent-assisted reconciliation pass for the ~20% of hospitals that don't match cleanly.

**Status:** scaffold. No functionality yet. See milestone M4.
